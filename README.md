# TogetherEat

### 이 프로젝트는 무엇인가요?
하나의 프로젝트에 다양한 분야(Web, Server, Android, AI, Design)의 기술을 구현하는 프로젝트 입니다.

### 프로세스
1. 먹고 싶은 음식사진을 찍거나 저장한다.
2. 그 이미지를 어플이나 웹에 올린다.
3. 머신러닝을 통해 그 음식이 무엇인지 파악한다.
4. 파악된 음식관련 단톡방에 초대된다.

### 사용기술
Front : React  
Back : Spring  
Android : Java  
Machine-Learning : Python
